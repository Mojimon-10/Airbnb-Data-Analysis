# Airbnb-Data-Analysis
Data analysis and machine learning project using an Airbnb dataset from Kaggle. Includes data cleaning, exploratory data analysis (EDA), visualization, and price prediction modeling in R.

## Tools Used
- R
- tidyverse
- ggplot2
- caret
- corrplot

## Objectives
- Clean and preprocess real-world Airbnb dataset.
- Perform exploratory data analysis.
- Visualize pricing trends across countries.
- Build predictive models for price estimation.

## Key Findings
- Bedrooms and bathrooms strongly influence price.
- Higher-rated listings tend to charge more.
- Country significantly impacts price distribution.

## Machine Learning Models
- Linear Regression.
